links locais, externos e donwload de arquivos

links são conhecidos com âncoras e permitem ao clientes acessarem outros contéudos.

Vamos primeiro acessar outro site que foi criado por nós na mesma pasta. → Clique aqui ← e acesse a segunda página que criamos!

Agora vamos entrar um uma terceira página criada.Está estará em uma outra pasta. → Clique aqui ← e acesse a terceira página.

Podemos também criar um link que subscreve(abandona) o nosso site para entrar em outro. Por exemplo, Acesse o YouTube(não se esqueça de voltar clicando no botão "VOLTAR").

Também Existe a opção de abrirmos o mesmo link em outro página em branco, conservando a atual. → Clique aqui ← e confira o resultado.

Trabalhando com Downloads

Os links podem ser usados também para fazer downloads. Vamos abrir um livro PDF por exemplo: Abra o livro aqui.

Você também pode baixar uma versão compactada(.zip).→ Clique aqui ← para fazer o download.
